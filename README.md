# Mythri Services Insurance

Static web app for iPlanet Care, Kochi - VYT2 insurance case tracking.

## Files

- `index.html` - main application
- `data.js` - VYT2 insurance case data
- `iplanet-care-logo.png` - iPlanet Care logo

## Hosting

Upload this folder as a static site. The app does not require a backend.

Recommended quick hosting:

- Netlify Drop: upload this folder or the zip file
- Vercel: import as a static project
- cPanel/shared hosting: upload all files into `public_html`

The app stores live edits in the browser and exports the current table to Excel/CSV.
